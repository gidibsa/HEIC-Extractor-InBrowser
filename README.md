# HEIF/HEIC Extractor and Converter

This is a learning project built in order to try my hands at NextJS. I will build on this knowledge to create other interesting projects.

## Features

This project was built using 
    - NextJS
    - TailwindCSS
    - ShadCn
    - LibHeif
    - Acertenity
    - Lucide
    - MagicUI
